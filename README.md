# hwaseong-compass

화성 살이 나침반 — 「AI 화성 챌린지」 시민 부문 출품작.

전입 시민이 직장·가구·차량 조건을 입력하면 화성 동네별(동탄 / 비봉·봉담 / 향남·우정) 통근시간 분포·국공립 보육 대기확률·생활비를 브라우저 내 몬테카를로 시뮬레이션으로 생성해 비교한다. 정책 모드는 같은 엔진으로 보육시설 미충족 권역의 우선순위를 진단한다. 서버 없이 단일 HTML로 동작.

GitHub Pages: `Settings → Pages → Source: GitHub Actions`를 한 번 켜면 `index.html`이 자동 배포된다.
